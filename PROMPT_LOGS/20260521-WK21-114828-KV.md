Both answers:

**1. Scroll back in tmux:** `Ctrl-b [` enters copy/scroll mode. Then PageUp/PageDown (or arrow keys, or `Ctrl-u`/`Ctrl-d` for half-pages). Press `q` to exit. To search: `?` searches backwards, `/` searches forwards, `n`/`N` to next/prev match.

**2. The target you're thinking of is `make endpoints`** (Makefile:405) — "Show all key endpoints for ENV (ALB URL, deploy-host SSM, RDS, Valkey, FSx) — copy-paste friendly." That's the one that gives you the Drupal site URL plus everything else in one shot.

Two other relevant ones if you want just the admin entry point:
- `make admin-login-url` (Makefile:2311) — generates a one-time drush user:login URL for admin
- `make smoke-test-drupal` (Makefile:2314) — curls the ALB with the Drupal Host header
