**What needs to come together for end-to-end testing:**

1. **A vhost file on FSx** at `/etc/nginx/shared/sites-enabled/drupal.conf` containing:

   - `root /var/www/sandbox/drupal/web;`
   - The Drupal location blocks (`/` → try_files → index.php, FastCGI to `php83`)

2. **Drupal's `trusted_host_patterns`** already accepts `drupal-sandbox.test` (DRUPAL_SITE_NAME) AND `*.elb.amazonaws.com` — so the Host header just needs to be one of those.

3. **/etc/hosts on Mac** mapping `drupal-sandbox.test` → ALB. Since ALB IPs aren't stable, either pin a current IP from `dig` (will break when ALB scales) or just use `curl -H "Host: drupal-sandbox.test" http://<alb-dns>/` for testing without /etc/hosts at all.
