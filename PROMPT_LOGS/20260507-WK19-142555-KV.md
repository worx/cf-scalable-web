## What to watch for in your first run

The install command should see clean output for these key moments:

1. `Installing drupal/core-recommended (...)` — Drupal core package fetch
2. `Generating optimized autoload files` — composer's autoloader build
3. `Installing Drupal 11.x.x` — drush kicking off the install
4. `Drupal Composer Project version : ...` — final drush status check

If any of those error out, screenshot or paste the relevant 20-30 lines and we'll fix in seconds. After this works, building `install-drupal.sh` (cloud) is mostly swapping the db-url and install path.

Run `make install-drupal-local` whenever you're ready.
