### 10:18:37 - Response

Since the mount command is in the Image Builder templates (boot scripts baked into AMIs), you need to rebuild the AMIs. From the deploy host:

```bash
cd ~/projects/cf-scalable-web
git pull
make deploy-image-builder ENV=sandbox
make upload-build-configs ENV=sandbox
make build-amis ENV=sandbox
```

Wait for the builds (~25 min), then:

```bash
make update-ami-param ENV=sandbox PIPELINE=nginx
make update-ami-param ENV=sandbox PIPELINE=php74
make update-ami-param ENV=sandbox PIPELINE=php83
```

Then terminate the current instances and let the ASG launch fresh ones with the new AMIs:

```bash
make deploy-compute-nginx ENV=sandbox
make deploy-compute-php ENV=sandbox
```

Or just terminate the instances manually from the console — the ASG will relaunch with the updated AMI parameters.
