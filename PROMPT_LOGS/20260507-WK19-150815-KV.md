## What's in `make verify-drupal-local`

Single command that confirms:
- ✅ `drush status` reports version, db driver, bootstrap, PHP version
- ✅ Admin user exists and is enabled
- ✅ Row counts for users / nodes / config / sessions tables
- ✅ Recent watchdog entries (or "(no log entries yet)" if none)

If all four return cleanly, the install is healthy.
