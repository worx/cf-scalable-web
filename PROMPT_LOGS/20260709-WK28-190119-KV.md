## Plan: `make migration-*` ‚Äî automated end-to-end runbook

### Design principles (baked in throughout)

- **Idempotent** ‚Äî re-run anything without breakup or duplicates
- **Composable** ‚Äî each step is standalone AND part of a group
- **Fail-loud** ‚Äî `set -euo pipefail` in every script, explicit precondition checks
- **Isolated from main Makefile** ‚Äî a `migration.mk` file `include`'d into the main Makefile keeps main lean
- **Two execution contexts** ‚Äî jumpbox scripts and deploy-host scripts stay in separate directories, never confuse the two
